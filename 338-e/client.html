3. Clients#

A client is used to send queries to Presto and receive results, or otherwise interact with Presto and the connected data sources.

Some clients, such as the command line interface, can provide a user interface directly. Clients like the JDBC driver, provide a mechanism for other tools to connect to Presto.

The following clients are available:

  • 3.1. Command Line Interface
  • 3.2. JDBC Driver

In addition, the community provides numerous other clients for platforms such as Python, and these can in turn be used to connect applications using these platforms.

SEP includes support and documentation for other client usage:
