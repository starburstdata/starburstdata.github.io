2.6. Microsoft Azure#

Microsoft Azure can be used in numerous ways to run Presto, and optionally the related data sources and infrastructure in the cloud:

  • Use the Kubernetes support and run it on Azure Kubernetes Service.

  • Install and run SEP on Azure compute instances leveraging the installation from the tar.gz archive.

  • Install and run SEP on Azure compute instances using the RPM package.

  • Use the Hive connector to query Azure Storage
