Release 336-e STS (19 Jun 2020)#

Starburst Enterprise Presto 336-e is the follow up release to the 332-e version.

  • Highlights

The 336-e release includes all improvements from the Presto project releases since Presto 334:

  • Presto 335

  • Presto 336

This release is a short term support (STS) release.

Highlights#

  • MapR Hive metastore support

  • Hive connector storage caching now fully supported

  • Preview version of the native table operator for the Teradata Parallel Direct connector
