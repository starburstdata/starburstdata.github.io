10.1. Starburst Connectors Overview#

SEP includes numerous connectors. These connectors are either extensions of Presto connectors adding features such as table statistics, user impersonation and others or completely separate additional connectors for other data sources.

Connectors and connector extensions, that do not require a license:

  • Starburst Generic JDBC Connector

  • Starburst Hive Connector

    • Starburst Hive Connector Security

    • IBM Cloud Object Storage Support for the Hive Connector

  • Starburst Kafka Connector

  • Starburst MySQL Connector

  • Starburst PostgreSQL Connector

  • Starburst SQL Server Connector

Connectors and connector extensions, that require a license:

  • Starburst IBM DB2 connector

  • Starburst Delta Lake connector

  • MapR Hive support

  • Starburst Oracle connector

  • Starburst Snowflake connector

  • Starburst Teradata connector

  • Starburst Teradata Parallel Direct connector

Connectors with support for table statistics that can be used by the cost-based optimizer:

  • Starburst IBM DB2 connector

  • Starburst Hive Connector

  • Starburst MySQL Connector

  • Starburst Oracle connector

  • Starburst PostgreSQL Connector

  • Starburst Snowflake connector

  • Starburst SQL Server Connector

  • Starburst Teradata connector
