5.1. Overview#

The Kubernetes support for Starburst Enterprise Presto® (SEP) allows you to run your SEP clusters and additional components such as your Hive metastore or Apache Ranger. The features of k8s allow you to efficiently create, operate and scale your clusters and adapt them to your workload requirements.

Kubernetes platforms#

The following Kubernetes cluster services are supported:

  • Amazon Elastic Kubernetes Service (EKS)

  • Google Kubernetes Engine (GKE)

  • Microsoft Azure Kubernetes Service (AKS)

  • Red Hat OpenShift

  • Private Kubernetes installations

This chapter aims to cover all the above, but focussed on the generic details around k8s usage that apply to all services.

Features#

The following Helm charts are available as part of the k8s offering:

  • Starburst Enterprise Presto® (SEP) for coordinator and worker nodes

  • Apache Ranger with Starburst Ranger plugin and LDAP user synchronization and PostgreSQL database to support system level security support of Starburst Enterprise Presto

  • Apache Hive metastore
