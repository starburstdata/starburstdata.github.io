1. Overview#

Presto is a distributed SQL query engine designed to query large data sets distributed over one or more heterogeneous data sources.

Presto is maintained by the large community of contributors and users of the Presto Software Foundation.

  • 1.1. Use Cases
  • 1.2. Presto Concepts

Starburst Enterprise Presto (SEP) is the commercial distribution of Presto. It is is maintained by Starburst and free to download and use. It includes additional security features, more connectors, a cost-based query optimizer, support for running additional deployment platforms and much more.

Download Starburst Enterprise Presto now!

A number of enterprise features require a license key to be enabled.

Connectors and related extensions:

Most connectors and related extensions are documented in Starburst Connectors Overview.

Security:

  • Event Logger

  • Query audit

  • Okta authentication

  • Role Based Access Control

    • System level security with Apache Ranger

    • Hive level security with Apache Ranger

    • Hive level security with Apache Sentry

All security features require a license.

Operational

  • AWS and CFT support

  • Kubernetes support

  • Using large pods in K8s operator

  • Usage metrics

More information about obtaining and installing a license can be found in Starburst Enterprise Presto license.
