    Verify server with Web UI #

    To verify that your locally run Starburst Enterprise server is running, use the Web UI.

    Open the Web UI #

    With any modern browser, go to http://localhost:8080.

    At the login screen, enter your current OS login name (or any string).

    The default Web UI screen shows the version number, environment, and uptime of the server. The statistics fields show zeros until a query is run against the server.

    Run queries #

    To run queries against the local server with the Trino CLI, see CLI.
