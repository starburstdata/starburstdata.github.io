Release 422-e STS (20 Jul 2023)#

Starburst Enterprise platform (SEP) 422-e is the follow up release to the 420-e version. It includes all improvements from the following Trino releases:

  • Trino 421

  • Trino 422

This release is a short term support (STS) release.

Breaking changes#

  • The preferredWritePartitioningMinNumberOfPartitions cache service configuration property and underlying functionality has been removed. You must remove this configuration property or table scan redirection fails.

Security#

  • Added support for SHOW CREATE statements in AWS Lake Formation.

Delta Lake connector#

  • Added support for Ceph storage as a backend object storage.

  • Added support for Dell ECS and ObjectScale as backend object storage.

Hive connector#

  • Added support for Ceph storage as a backend object storage.

  • Added support for Dell ECS and ObjectScale as backend object storage.

Iceberg connector#

  • Added support for Ceph storage as a backend object storage.

  • Added support for Dell ECS and ObjectScale as backend object storage.

Snowflake connector#

  • Added support for the snowflake_parallel connector.
