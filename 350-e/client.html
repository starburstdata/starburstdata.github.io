Clients#

A client is used to send queries to Starburst Enterprise platform (SEP) and receive results, or otherwise interact with SEP and the connected data sources.

Some clients, such as the command line interface, can provide a user interface directly. Clients like the JDBC driver provide a mechanism for other tools to connect to SEP.

The following basic clients are available:

  • Command line interface
  • JDBC driver

In addition, the community provides numerous other clients for languages such as Python or Go, and these can in turn be used to connect applications based on those languages.

SEP includes support and documentation for the following additional clients:

  • Starburst JDBC driver
  • Starburst ODBC driver
  • Tableau
  • Starburst Power BI DirectQuery driver
