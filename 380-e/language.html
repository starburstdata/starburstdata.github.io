SQL language#

You’ve come to the right place if you are looking for reference material on SQL in Trino. If you want in-depth training on handling complex data types and more, check out our advanced SQL video training. It covers topics such as:

  • using JSON

  • array and map functions

  • advanced aggregations

  • window functions

View the training!

SEP uses the Trino SQL query engine that allows users to use standard SQL, and there user their favorite data tools, including BI and ETL tools, with any underlying data source.

This chapter provides an overview of SQL support, a reference to the supported SQL data types and other general characteristics of the SQL support in SEP.

A complete SQL statement and syntax reference and documentation on numerous SQL functions and operators are available in separate chapters.
