Release 376-e STS (21 Apr 2022)#

Starburst Enterprise platform (SEP) 376-e is the follow up release to the 375-e version. It includes all improvements from the following Trino project release:

  • Trino 376

This release is a short term support (STS) release.

Breaking changes#

  • The ldap.ssl-trust-certificate configuration property for an LDAP password authenticator has been deprecated in favor of ldap.ssl.keystore.path and ldap.ssl.truststore.path for two-way TLS/SSL with an LDAP authentication server. Clusters that use the deprecated property in their LDAP password authenticator configuration fail to start.

Security#

  • Allow leaving email address and last name empty when creating users in Ranger with SEP user synchronization.

Netezza connector#

  • Add support for JDBC connection pooling
