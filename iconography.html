    Iconography #

    This page describes the common iconography and visual language used throughout Starburst technical content about Trino, Starburst Enterprise, Starburst Galaxy, and related components and use cases.

    Clusters and cluster components #

    Trino-based clusters and cluster components are displayed in pink by default. When two clusters must be shown together, teal is used for any additional Trino-based clusters.

    Note: Any cluster that is not Trino-based is depicted in black.

    The following are miscellaneous icons in their default colors. These may be paired with the logo to clearly identify a specific platform or technology. Secondary color choices are sometimes used to visually differentiate like items.

    Security-related #

    Security components available as a part of Starburst Galaxy or Starburst Enterprise are always depicted in teal. Third-party security components are always depicted in black.
