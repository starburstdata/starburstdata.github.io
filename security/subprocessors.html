    Starburst Subprocessors #

    Starburst Galaxy uses the following third-party subprocessors to assist in providing services. The subprocessors involved depend on the services ordered as outlined in the terms of service and license agreements.

    FullStory #

    FullStory provides a web-based digital intelligence system to assess Starburst Galaxy customer activity.

    Data #

    Data accessed by the subprocessor:

    • Email address
    • User ID

    Transfer mechanism #

    SCCs (standard contractual clauses)

    Supplemental measures #

    Encryption of data at rest and in transit.

    For more information, refer to the following FullStory articles:

    • Security overview
    • GDPR FAQs

    Contact details #

    FullStory Incorporated
    1745 Peachtree Rd NW Suite G
    Atlanta, GA 30309
    USA

    Email: privacy@fullstory.com
