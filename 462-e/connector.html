Connector overview#

This section describes the connectors available in Starburst Enterprise to access data from different data sources by configuring catalogs with the connector-specific properties in catalog properties files.

  • Object storage - Covers all Starburst-supported object storage and metastores.

  • Non-object storage - Covers all Starburst-supported non-object storage, including the generic JDBC connector.

  • Community connectors - Covers connectors supported by the Trino community.

  • Utilities - Includes system information tools, and the smart index and caching tool.

The Starburst connectors feature matrix contains details about the key features, license requirements and other aspects about all the connectors included with SEP.
