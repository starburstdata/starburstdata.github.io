Release 392-e STS (8 Aug 2022)#

Starburst Enterprise platform (SEP) 392-e is the follow up release to the 391-e version. It includes all improvements from the following Trino project releases:

  • Trino 392

This release is a short term support (STS) release.

General#

  • Fix issue with live queries being filtered incorrectly.

  • Fix a link that redirects to Ranger in the Starburst Enterprise web UI.

  • Add API support to force publish a data product. This operation recreates all the datasets of already published data products.

Security#

  • Fix incorrectly displayed Built-in access control privileges created through SQL for a schema.

Starburst Redshift connector#

  • Improve support for reading and writing to external tables.

Teradata connector#

  • Add predicate pushdown support for SQL statements using IN.

  • Add procedure system.flush_metadata_cache() to flush metadata caches.
