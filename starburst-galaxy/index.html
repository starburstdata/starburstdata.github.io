Starburst Galaxy

Starburst Galaxy is a fully-managed data lake analytics platform built on top of Trino. It takes care of the heavy lifting of designing, provisioning, maintaining, and securing your data infrastructure while making it easy to discover, govern, and consume your data.

    Get started #

    • Sign up, or request access from your Starburst Galaxy administrator.
    • Configure catalogs for your data sources.
    • Use the catalog in a cluster.
    • Query with SQL statements in the query editor or another client application.
    • Learn how roles and privileges work to protect your data in the access control overview.

    Gravity #

    What is Starburst Gravity?

    Gravity is a unified governance layer that lets you manage more than just the data in the assets connected to Starburst Galaxy, but also manage access to and gain insights about that data.

    Gravity is a set of interconnected Starburst Galaxy features:

    Universal search

    Locate any data object anywhere throughout your Starburst Galaxy account.

    Catalog explorer

    Explore metadata about your data sources at the catalog, schema, and table levels.

    Schema discovery

    Automatically identify and register new tables and views added to object storage catalogs.

    Data governance
      Roles and privileges

      Assign privileges to roles, then assign roles to individual users for thorough, safe role-based access control (RBAC).

      Tags and policies

      Assign attribute tags to catalogs, schemas, tables, views, or columns, then create attribute-based policies to manage fine-grained access control (ABAC).

      Single sign-on and SCIM

      Configure your site's SAML-based single sign-on (SSO) identity provider such as Okta to manage Galaxy logins. If your SSO provider supports SCIM, allow it to manage group and usernames for Galaxy.

    Data products

    Data products provide a collection of curated, high-quality related datasets and relevant metadata that increase the discoverability of your data

    Query sharing

    Share any query editor tab with other roles to allow collaborative development of complex queries.
