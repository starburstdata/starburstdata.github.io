Starburst Galaxy is the cloud native and fully managed service of Starburst’s massively parallel processing (MPP), highly performant query engine. It allows you to query a variety of data sources, or join data across multiple data sources through a single query.

    Get started #

    • Sign up, or request access from your Starburst Galaxy administrator.
    • Configure catalogs for your data sources.
    • Use the catalog in a cluster.
    • Query with SQL statements in the query editor or another client application.
    • Learn how roles and privileges work to protect your data in the security overview.
