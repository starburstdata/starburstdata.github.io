    PostgreSQL data source #

    You can connect PostgreSQL databases in SGP as a data source.

    Requirements #

    • User with sufficient access to the data source must be configured with a secret.
    • Established private connection between SGP and PostgreSQL, , or inbound and outbound security groups to allow 0.0.0.0/0 for all traffic.

    Operation #

    • Add a PostgreSQL data source
