Query optimizer#

The following reference documents provide details on SEP’s powerful query optimization tools. The information helps you to understand how Starburst Enterprise platform (SEP) achieves higher query performance out of the box, and how to further improve SQL query performance.

The following pages describe how SEP works to efficiently process queries:

  • Table statistics
  • Pushdown
  • Cost based optimizations

Learn how to read an EXPLAIN plan to help you to understand costs and improve the performance of your queries:

  • Cost in EXPLAIN

Starburst offers configuration properties to help you refine query performance even further:

  • Starburst optimizer properties
