Release 347-e STS (30 Nov 2020)#

Starburst Enterprise Presto 347-e is the follow up release to the 346-e version. It includes all improvements from the following Presto project releases:

  • Presto 347

This release is a short term support (STS) release.

Security#

  • Add support to CREATE, ALTER, and DROP resources in Ranger

  • Avoid query failures when impersonation is used with Presto Ranger plugin policies in Privacera Platform

  • Add new Starburst Ranger CLI for service definition and user management in Ranger

Delta Lake connector#

  • Avoid skipping of transaction log entries after external data updates

Generic JDBC connector#

  • Add support for dynamic filtering

Kafka connector#

  • Fix SSL/SASL authentication support

SAP HANA connector#

  • Add aggregate function pushdown

Teradata connector#

  • Support ALTER TABLE statement usage

Known issues#

  • A library update for the Presto CLI caused rendering issues of the query processing progress display. No functional problems occur. A workaround is to use the Presto CLI from an earlier release.
