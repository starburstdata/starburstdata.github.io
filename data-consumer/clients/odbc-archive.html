    ODBC driver archive #

    It is strongly recommended that you use the latest stable version of the Starburst ODBC driver (v2.1.1.1002) to connect to Starburst Galaxy or SEP.

    Note: If opening the macOS ODBC driver package causes a malicious software warning to appear, this warning can be safely ignored. Read Apple support documentation for information on how to bypass this warning message.

    In addition to the latest stable driver version, this page includes download and documentation links for other versions of the Starburst ODBC driver. This includes engineering release drivers, which may not have full documentation or downloads for some operating systems. You should only use an engineering release of the ODBC driver as directed by Starburst Support.
