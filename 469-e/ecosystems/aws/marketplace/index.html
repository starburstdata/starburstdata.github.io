Starburst Enterprise in AWS Marketplace#

Starburst Enterprise platform (SEP) is available directly through Amazon AWS Marketplace to run on a variety of instance types.

You can deploy SEP using:

  • AWS Marketplace EKS for Kubernetes and container-based deployments
  • AWS Marketplace CFT for legacy deployments on EC2
