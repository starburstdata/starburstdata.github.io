Starburst Enterprise 405-e STS

Jan 3, 2023 | Joe Lodin , Manfred Moser

Starburst Enterprise platform (SEP) 405-e STS has been released as a follow-up to the 403-e STS release.

The release is based on Trino 405, which includes the following noteworthy changes:

  • Support for ALTER COLUMN ... SET DATA TYPE statement.
  • Support for Apache Arrow when reading from BigQuery.
  • Support for views in Delta Lake connector.
  • Support for the Iceberg REST catalog.
  • Support for Protobuf encoding in Kafka connector.
  • Support for fault-tolerant execution in MongoDB connector.
  • Support for DELETE and query pushdown in Redshift connector.
  • Performance improvements when reading Parquet data.

On the SEP side we included a lot of minor improvements and bug fixes for smart indexing and caching. For more information, refer to the release notes for 405-e.

Joe and Manfred
