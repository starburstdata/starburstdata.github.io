Starburst cluster sizing, and performance tuning

Presenters: Dain Sundstrom and Manfred Moser
Video date: 9 September 2020
Running time: 2h 15m


This training session is geared towards helping users tune and size their deployment for optimal performance. Delivered by Dain Sundstrom, this session covers the following topics:

  • Cluster configuration and node sizing
  • Memory configuration and management
  • Improving task concurrency and worker scheduling
  • Tuning your JVM configuration
  • Investigating queries for join order and other criteria
  • Tuning the cost-based optimizer
