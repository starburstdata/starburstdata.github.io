Understanding and tuning Starburst query processing

Hosts: Martin Traverso, Manfred Moser
Video date: 12 August 2020
Running time: 2h09m


This training session is geared towards helping users understand how Starburst executes queries. That knowledge can help you improve query performance. For instance, the explain plan is a powerful tool. We explore how to access the explain plan and how to read it. We look at the work the cost-based optimizer performs and how you can potentially help run your queries even faster. Delivered by Martin Traverso, this session covers the following topics:

  • Explain the EXPLAIN
  • Learn how queries are analyzed and executed
  • Understand what the optimizer does, including some of its limitations
  • Showcase the cost-based optimizer
