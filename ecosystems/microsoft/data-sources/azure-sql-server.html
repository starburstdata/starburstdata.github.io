    SQL Server and Azure SQL #

    You can access Microsoft SQL Server and Azure SQL data sources in Microsoft Azure with Starburst Enterprise platform (SEP) and Starburst Galaxy. Microsoft offers numerous variants of SQL Server on Azure:

    • Azure SQL Database
    • Azure SQL Managed Instance
    • SQL Server on Azure Virtual Machines
    • Azure SQL

    Starburst Enterprise #

    Use the Starburst SQL Server connector in your catalogs to query databases.

    Starburst Galaxy #

    Use a SQL Server catalog to query databases in Azure SQL Database.
