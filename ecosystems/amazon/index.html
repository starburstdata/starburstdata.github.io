    Amazon AWS #

    Starburst Enterprise platform (SEP) can be used with many components of Amazon AWS. You can deploy there, access data sources in Amazon AWS and use some of the other features of Amazon AWS.

    Find all the relevant information for using Starburst and Amazon AWS in the following sections and guides.

    Deployments #

    You can install Starburst on Amazon Elastic Kubernetes Service (EKS), Amazon Elastic Compute Cloud (EC2) or through the AWS Marketplace, once you have established your enterprise account.

    To get started Starburst offers a trial license and help with your proof of concept.

    Amazon EKS #

    Amazon Elastic Kubernetes Service (EKS) is certified to work with SEP.

    More information is available in our dedicated EKS section.

    Amazon EC2 #

    As a user of Amazon EC2 instances you can manage your SEP cluster with the same AMI and CloudFormation Template as AWS Marketplace users.

    Alternatively, you can manage a RPM or tarball-based installation with Starburst Admin.

    AWS Marketplace #

    SEP is available on AWS marketplace via CloudFormation Template

    Documentation #

    You are currently viewing the AWS section of our user guides. In the left-hand navigation are guides for AWS-specific topics. We also have user guides tailored toward the different types of Starburst users: data consumers, data engineers and platform administrators.

    Our comprehensive reference documentation is available for all supported releases.

    Support #

    No matter where you deploy from, Starburst Support has your back.

    Enterprise accounts #

    Starburst enterprise customers enjoy:

    • Dedicated resources for your organization
    • 24/7 expert support
    • Rapid-response SLAs
    • Access to knowledge base articles and an intuitive support portal

    AWS marketplace customers #

    Starburst offers the following support for our marketplace subscribers without an enterprise contract:

    • Email-only support
    • Five email issues to awssupport@starburstdata.com per month
    • First response SLA of one business day
    • Support hours of 9 AM - 6 PM US Eastern Time
