    Starburst ODBC driver#

    The ODBC driver allows any client application with ODBC support to connect to Presto. Once connected, SQL statements can be used to query catalog, and therefore the underlying data sources.

    You must meet the minimal system requirements for installing the ODBC drivers on Windows, Mac, or Linux. Operating system specific packages and installers are available.

    Detailed information is available in the comprehensive ODBC driver documentation.

    Installation#

    The driver can be downloaded from the Starburst support site.

    The ODBC driver requires a license –– contact Starburst support for your license file.

    Run the ODBC installer. Note the installation path /Library/starburst/prestoodbc/ on MacOS. The driver documentation contains more details, including paths for different platforms.

    Copy your license (.lic) file to the lib directory in the ODBC driver installation, for example /Library/starburst/prestoodbc/lib/.

    Supported platforms#

    Windows

    • Windows 7, 8.1, or 10

    • Windows Server 2008 R2 SP1, 2012, or 2016

    MacOS

    • OSX version 10.9, 10.10, or 10.11

    • iODBC 3.52.7 or later

    Linux

    • Red Hat® Enterprise Linux® (RHEL) 6 or 7

    • CentOS 6 or 7

    • SUSE Linux Enterprise Server (SLES) 11 or 12

    • Debian 7, 8, or 9

    • Ubuntu 14.04 or 16.04

    • iODBC 3.52.7 or later

    • unixODBC 2.3.0 or later
