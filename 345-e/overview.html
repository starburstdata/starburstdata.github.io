    Overview#

    Presto#

    Presto is a distributed SQL query engine designed to query large data sets distributed over one or more heterogeneous data sources.

    Presto is maintained by the large community of contributors and users of the Presto Software Foundation.

    • Use cases
    • Presto concepts

    Starburst Enterprise Presto#

    Starburst Enterprise Presto (SEP) is the commercial distribution of Presto. It is is maintained by Starburst and free to download and use. It includes additional security features, more connectors, a cost-based query optimizer, support for running additional deployment platforms and much more.

    Download Starburst Enterprise Presto now!

    A number of enterprise features require a license key to be enabled. For more information about obtaining and installing a license, see Starburst Enterprise Presto license.

    This following feature lists are designed to help you understand the similarities and differences between open source Presto, and Starburst Enterprise Presto across several categories: platform, deployment scenarios, management, clients, security, support, and connectors.

    Connectors#

    SEP contains many additional connectors, as well as improvements of Presto connectors to increase performance, add security features and others. A detailed feature comparison is available in the SEP connectors overview.

    Clients#

    Clients are anything that connects to Presto to enable users of that tool to run SQL queries.

    For more information about connectors and related extensions, see SEP connectors overview.
