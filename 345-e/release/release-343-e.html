Release 343-e STS (28 Sep 2020)#

Starburst Enterprise Presto 343-e is the follow up release to the 341-e version. It includes all improvements from the following Presto project releases:

  • Presto 343

  • Presto 342

This release is a short term support (STS) release.

General#

  • Add Starburst MemSQL connector with table statistics, aggregate push down and more features as extension of MemSQL Connector

  • New and exclusive Starburst SAP HANA connector

  • Improved Policy management with SQL

Hive connector#

  • Allow specifying behavior if data is inserted into existing Hive partition with the hive.insert-existing-partitions-behavior property.

Snowflake connector#

  • Add aggregate function push down

  • Eliminate pushed down filters above table scan

Teradata connectors#

  • Eliminate pushed down filters above table scan

  • Fix illegal state exception related to nanosleep in table operator
