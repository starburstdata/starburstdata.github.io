Version 334-e-aws-ami-0.93 and 334-e-ami-0.93 (4 June 2020)#

  • Update to Starburst Enterprise Presto 334-e
  • Add more EC2 instance type choices

Version 332-e.3-aws-ami-0.92 and 332-e.3-ami-0.92 (28 May 2020)#

  • Update to Starburst Enterprise Presto 332-e.3

Version 332-e.2-aws-ami-0.91 and 332-e.2-ami-0.91 (7 May 2020)#

  • Update to Starburst Enterprise Presto 332-e.2
  • Create /data directory on Presto coordinator node for caching
  • Improved descriptions for numerous parameters

Version 332-e.1-aws-ami-0.90 and 332-e.1-ami-0.88 (30 April 2020)#

  • Update Starburst Enterprise Ranger to version 2.0.24
  • Add Amazon regions for Asia Pacific, Canada and South America
  • Use presto_service user from Superset to enable usage with active Ranger integration

Version 332-e.1-aws-ami-0.85 and 332-e.1-ami-0.84 (17 April 2020)#

  • Update to Starburst Enterprise Presto 332-e.1
  • Enable usage metrics collection
