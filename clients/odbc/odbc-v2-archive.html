    ODBC V2 driver archive #

    This page includes download links and documentation for the supported releases of the Starburst ODBC V2 drivers.

    Note: Starburst strongly recommends using the ODBC V3 drivers. Links to the V2 drivers on this page will be removed in 2026.

    This page includes engineering release drivers. Do not use an engineering release of the ODBC driver except as directed by Starburst Support.

    Note that opening the macOS ODBC driver package can cause a malicious software warning to appear. You can safely ignore this message. Read Apple support documentation for information on how to bypass this warning message.
