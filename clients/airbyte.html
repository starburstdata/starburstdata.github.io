    Airbyte #

    Airbyte is an open source data integration engine that helps you consolidate the data in your data warehouses, data lakes, and databases. You can use it to access clusters in Starburst Galaxy.

    For requirements and connection instructions, see the Starburst Galaxy destination user guide in the Airbyte documentation.
